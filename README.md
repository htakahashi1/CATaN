# CATaN

This repository contains the code for Canonical correlation Analysis of Transcriptome and TF-gene regulatory Networks (CATaN).

## Getting Started

1. Download the data and code directories.
2. Install the required packages.

## Usage

If you would like to apply CATaN to bulk RNA-seq data, you can either use the CATaN R package or the Jupyter Notebook code in the bulk directory. If you would like to apply it to single-cell RNA-seq data, use the code in the `scRNA-seq` directory.

Follow the numbers at the beginning of each Jupyter Notebook filename (e.g., **1**.code_name.ipynb). If you would like to use the CATaN R package, please refer to the README in the [CATaN](./CATaN) directory.
