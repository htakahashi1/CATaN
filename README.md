# CATaN
This repository include some codes for CATaN.

## How to start?
1. Download the data directory and code directory.
2. install the necessary packages.

## How to use?
If you would like to apply CATaN to bulk transcriptome data (ex. RNA-seq and microarray), use the codes in bulk directory. If you would like to apply it to single cell RNA-seq data, use the codes in scRNA-seq directory.
Follow the number at the the head of the jupyter notebook filename (ex. **1**.code_name.ipynb).
